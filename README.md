# Sales Dashboard - Power BI

## Project Overview
An interactive 2-page Sales Dashboard built using Power BI to analyze retail sales performance.

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- CSV Dataset

## Dashboard Pages
### Page 1 - Overview
- KPI Cards: Total Sales, Profit, Quantity, Profit Margin %, Avg Order Value
- Sales Trend by Year (Line Chart)
- Sales by Category (Bar Chart with Drill Down)
- Region Slicer (Interactive Filter)

### Page 2 - Product Analysis
- Sales by Sub-Category (Bar Chart with Conditional Formatting)
- Sales by Category (Pie Chart)
- Region Slicer

## Key Insights
- Technology category drives the highest sales
- South region has the highest number of orders
- Profit Margin is 15.29% overall

## Key Features
- DAX Measures (Profit Margin %, Avg Order Value)
- Drill Down from Category to Sub-Category
- Conditional Formatting on charts
- Interactive Slicers
